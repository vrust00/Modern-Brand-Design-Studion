##  Что за проект
Это верстка адаптивного многостраничного сайта для студии дизайна по макету из Figma из открытых источников. 
Включает главную страницу, страницу с примером одного из проектов и страницу «О студии».

##  Какие технологии использовал
- HTML5
- CSS3 (Flexbox, Grid)
- CSS Анимации
- Figma
- JavaScript (меню)

##  Что получилось
- Полная адаптация под мобильные устройства
- Семантическая разметка
- Меню, ссылки — всё работает, можно проверить в [ДЕМКЕ](https://vrust00.github.io/Modern-Brand-Design-Studio/)


##  Как запустить у себя
1. Склонируйте репозиторий (cmd):  
   `git clone https://github.com/vrust00/Modern-Brand-Design-Studio.git`
2. Откройте файл `index.html` в любом браузере.
